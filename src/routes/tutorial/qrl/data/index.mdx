Оптимизатор можно использовать и для ленивой загрузки данных, а не только функций. Преимуществом этого является то, что ленивая загрузка любой части вашего приложения становится простой.

Используйте отдельную функцию `$()` для пометки данных, которые вы хотите загрузить лениво. Функция `$()` возвращает `QRL`-адрес, который является ссылкой и может быть сериализован Qwik, а впоследствии может быть восстановлен в исходное значение.

### Пример

Лениво загрузите данные, представляющие сообщение, как часть слушателя клика.

Представьте, что в этом упражнении мы хотим лениво загрузить строку `'Привет, мир!'`. Используйте функцию `$()`, чтобы пометить строку как лениво-загружаемую. Затем используйте `await` для разрешения значения.

Перейдите на вкладку `Symbols` и изучите, как код был разбит на части.

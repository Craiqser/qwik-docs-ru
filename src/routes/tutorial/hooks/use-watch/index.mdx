Используйте `useWatch$()` для выполнения функции перед начальным рендером и при каждом изменении отслеживаемых значений. Функция выполняется перед рендером, но она не может задержать рендер, поэтому если `useWatch$()` является асинхронной, рендер произойдет до того, как `useWatch$()` будет полностью выполнена.

## Отслеживание изменений в хранилище

Хук `useWatch$()` получает функцию `track()`, используемую для настройки подписок, которые будут автоматически повторять хук `useWatch$()`. Набор свойств `track()` сбрасывается при каждом выполнении `useWatch$()`, поэтому важно всегда использовать `track()` для настройки подписок заново. Из-за этого `useWatch$()` может подписываться на разные свойства с течением времени.

## Очистка

Хук `useWatch$()` может вернуть функцию обратного вызова для очистки. Это полезно для очистки любых ресурсов перед новым выполнением.

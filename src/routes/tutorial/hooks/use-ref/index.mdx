Используйте `useRef()` для получения информации об элементах DOM, созданных компонентом.

Получение ссылки на элемент DOM состоит из двух частей:

1. `useRef()` возвращает объект `ref` со свойством `current`, которое будет содержать ссылку.
2. `ref={_ref_variable_}` - это привязка параметра, которая установит свойство `current` объекта `ref` на элемент DOM.

### Пример

В примере справа `useRef()` используется для получения ссылки на элемент `input`. Однако в нём отсутствует привязка к параметру `ref`. Добавьте привязку к параметру `ref`, чтобы пример работал так, как ожидается.

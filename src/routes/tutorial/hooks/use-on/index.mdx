Используйте `useOn()` / `useOnDocument()` / `useOnWindow()` для программной установки слушателей на элементы хоста. Это полезно, когда вы создаёте пользовательские API и не имеете доступа к размещению этих событий в JSX или если события не известны заранее, например, создаются на основе реквизитов компонента.

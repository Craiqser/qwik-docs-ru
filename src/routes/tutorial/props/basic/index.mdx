Веб-приложения строятся из компонентов так же, как обычные приложения строятся из функций.

Композиция функций имела бы мало смысла, если бы вы не могли передавать в них параметры. Точно так же, как у функций есть параметры, у компонентов есть «пропсы». Компонент использует их для передачи данных своим дочерним компонентам.

> **Ваша задача**: Измените код так, чтобы родительский компонент `<App>` мог передать `salutation="Привет"` и `name="мир"` в компонент `<Greeter>`. Добавьте свойства в интерфейс `GreeterProps`, а затем добавьте необходимые привязки.

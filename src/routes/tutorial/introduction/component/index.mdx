Добро пожаловать в учебник Qwik. Это серия коротких уроков, которые помогут вам начать работу с Qwik. Введение представляет собой набор задач, которые научат вас создавать компонент, сохранять данные, подключать слушателей и получать данные с сервера. Итак, давайте начнём.

## Создание компонента

Используйте редактор справа, чтобы создать пользовательский интерфейс, который выглядит примерно так:

```
GitHub-организация: __________
- Qwik
- Partytown
```

HTML-код для этого компонента должен быть таким:

```HTML
<div>
  <span>
    GitHub-организация:
    <input value="BuilderIO" />
  </span>
  <div>
    <ul>
      <li>
        <a href="https://github.com/BuilderIO/qwik">Qwik</a>
      </li>
      <li>
        <a href="https://github.com/BuilderIO/partytown">Partytown</a>
      </li>
    </ul>
  </div>
</div>
```

Вы можете нажать кнопку "Решить" в левом нижнем углу, чтобы увидеть готовое решение.

В этом уроке мы добавим слушателя событий для элемента `<input>`. Для этого мы будем использовать событие `keyup`. Цель состоит в том, чтобы обновлять `org` по мере того, как пользователь набирает текст.

Используйте свойство `onInput$` для элемента `<input>`, чтобы добавить слушателя событий.

Обратите внимание, что присвоение хранилищу (`github.org = ...`) автоматически вызывает повторный рендер компонента.

В предыдущем разделе мы обсуждали, как можно лениво загружать стили по мере необходимости с помощью хука [`useStyles$()`](/docs/components/styles/index.mdx#usestyles)`. Стили браузера являются глобальными и применяются ко всем элементам DOM, по этой причине Qwik также предоставляет способ загрузки стилей, специфичных для конкретного компонента. Это достигается путем создания уникального класса для каждого компонента и последующей вставки этого уникального идентификатора класса в таблицу стилей.

Используйте `useStylesScoped$()` для загрузки и применения стиля только к определённому компоненту.

### Пример

В этом примере есть два компонента, и они оба имеют класс с одинаковым именем. Из-за этого происходит коллизия двух стилей, что приводит к нежелательному поведению. Используйте хук `useStylesScoped$()`, чтобы применить стили к определенному компоненту и исправить коллизию.

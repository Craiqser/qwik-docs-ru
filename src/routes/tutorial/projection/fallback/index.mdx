Содержимое по умолчанию позволяет дочернему компоненту отображать контент в случае, если родительский компонент его не предоставил. Содержимое по умолчанию может быть предоставлено с помощью CSS, для этого необходимо добавить селектор (и соответствующий CSS) в именованный слот, если он пуст.

### Пример

В этом примере мы показываем три карточки, но некоторые из них не заполнены. Используйте правильный селектор CSS, чтобы указать контент по умолчанию.

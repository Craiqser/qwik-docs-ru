Хранилища всегда должны быть сериализуемыми, однако иногда вы можете захотеть сохранить такое значение, которое не может быть сериализовано. Подобные значения сохраняются в браузере только при возобновлении работы приложения на клиенте.

Когда приложение приостанавливается, несериализуемые свойства отбрасываются.

> **Ваша задача**: Добавить функцию очистки в `store.cleanup`, чтобы кнопка `stop` могла очищать интервал с помощью `clearInterval()`.
> Для этого добавьте импорт `noSerialize` из '@builder.io/qwik' и используйте функцию `noSerialize` как обёртку для функции `clearInterval`.

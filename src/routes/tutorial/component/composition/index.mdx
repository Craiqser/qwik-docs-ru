Для создания приложений мы объединяем компоненты вместе.

В этом примере компоненты `<App />` и `<Greeter />` уже подготовлены для вас. Для этого упражнения поместите компонент `<Greeter />` внутрь компонента `<App />`, чтобы увидеть его содержимое.

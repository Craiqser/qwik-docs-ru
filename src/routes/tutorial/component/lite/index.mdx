Одна из суперспособностей Qwik заключается в возможности ленивой загрузки. Каждый компонент генерирует отдельный `Символ`, который загружается по запросу. Это позволяет создать большое приложение с большим количеством компонентов и загружать только те компоненты, которые необходимы для текущего представления.

В некоторых случаях вы можете захотеть загрузить компонент вместе с родительским компонентом. Например, если у вас есть компонент `<Greeter>`, который всегда используется с компонентом `<App>`, и вы хотите загрузить его как часть `<App>`. Это называется [встроенный компонент](/docs/(qwik)/components/overview/index.mdx#встроенные-компоненты).

В этом примере для вас подготовлены компоненты `<App>` и `<Greeter>`. Компонент `<Greeter />` объявляется с помощью метода [`component$()`](/docs/(qwik)/components/overview/index.mdx#component) и является компонентом Qwik. Удалите [`component$()`](/docs/(qwik)/components/overview/index.mdx#component), чтобы преобразовать `<Greeter>` во [встроенный компонент](/docs/(qwik)/components/overview/index.mdx#встроенные-компоненты).

Откройте вкладку `Symbols` и обратите внимание, что компонент `<Greeter />` больше не имеет независимого экспорта, а является частью компонента `<App>`.

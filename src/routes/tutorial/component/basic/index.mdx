Компоненты - это строительные блоки приложения Qwik. Компоненты объявляются с помощью [`component$()`](/docs/(qwik)/components/overview/index.mdx#component) и, как минимум, должны возвращать JSX-элемент.

Создайте компонент, который возвращает `Привет, мир!`.

# Промежуточное ПО для Netlify Edge

Промежуточное ПО Qwik City Netlify Edge позволяет подключить Qwik City к [Netlify Edge](https://docs.netlify.com/netlify-labs/experimental-features/edge-functions/).

## Добавление в Qwik-проект

Выполните эту команду, чтобы интегрировать netlify-edge в приложение:

```bash
npm run qwik add netlify-edge
```

Предыдущая команда создаст `src/entry.netlify-edge.tsx`, используя встроенное промежуточное ПО в приложении пользователя.

## Ручное добавление Netlify Edge

Если вы хотите добавить Netilify Edge вручную, вы можете добавить или изменить файлы, как показано ниже:

`src/entry.netlify-edge.tsx`

```typescript
import { qwikCity } from '@builder.io/qwik-city/middleware/netlify-edge';
import render from './entry.ssr';

const qwikCityHandler = qwikCity(render);

export default qwikCityHandler;
```

`vite.config.ts`

```typescript
import { defineConfig } from "vite";
import { qwikVite } from "@builder.io/qwik/optimizer";
import { qwikCity } from "@builder.io/qwik-city/vite";
import tsconfigPaths from "vite-tsconfig-paths";
+import netlifyEdge from "@netlify/vite-plugin-netlify-edge";

export default defineConfig(() => {
  return {
    plugins: [
      qwikCity(),
      qwikVite({
+        ssr: {
+          outDir: "netlify/edge-functions",
+        },
      }),
      tsconfigPaths(),
+      netlifyEdge({
+        functionName: "entry.netlify-edge",
+      }),
    ],
  };
});
```

Настройка `netlify.toml`

```toml
[[edge_functions]]
path = "/*"
function = "entry.netlify-edge"
```

## Использование

Используйте Netlify CLI для предварительного просмотра или развёртывания. [Полный текст руководства читайте здесь.](https://github.com/BuilderIO/qwik/blob/main/starters/servers/netlify-edge/README.md)

```bash
vite build --ssr src/entry.netlify-edge.tsx
netlify dev
```

### Контекст

[Netlify context](https://docs.netlify.com/edge-functions/api/#netlify-specific-context-object) доступен в параметре `platform` метода конечной точки:

```javascript
export const onRequest = async ({ platform }) => {
  platform.log('requested ip:', platform.ip)
};
```

Кроме того, вы можете импортировать тип `RequestHandlerNetlify`, чтобы иметь автокомплит типов в вашем редакторе.

```typescript
import {type RequestHandlerNetlify} from '@builder.io/qwik-city/middleware/netlify-edge';

export const onGet: RequestHandlerNetlify = (({platform}) => {
 //...
})
```

### Переменные окружения

```javascript
export const onRequest = async ({ platform }) => {
  platform.log('Vite env read from .dev file', import.meta.env.VITE_DEV)
  platform.log('netlify serverless env read from Netlify UI or CLI', Deno.env.toObject())
  return {}
};
```

# Промежуточное ПО для страниц Cloudflare

Вы можете добавить интеграцию со страницами Cloudflare с помощью этой команды:

```bash
npm run qwik add cloudflare-pages
```

Промежуточное ПО Qwik City Cloudflare Pages позволяет подключить Qwik City к [Cloudflare Pages](https://pages.cloudflare.com/).

Ниже приведен пример использования в пользовательском приложении.

```typescript
// Файл entry.cloudflare-pages.tsx
import render from './entry.ssr';
import { qwikCity } from '@builder.io/qwik-city/middleware/cloudflare-pages';

export const onRequest = qwikCity(render);
```

Скомпилированный модуль можно использовать в любом месте каталога страниц Cloudflare `/functions`.

```typescript
// Экспортируйте скомпилированный модуль туда, где его смогут найти страницы cloudflare.
// Например, используйте /functions/[[path]].ts или /functions/_middleware.ts
// чтобы Qwik City обрабатывал все запросы.

export { onRequest } from '../server/entry.cloudflare-pages';
```

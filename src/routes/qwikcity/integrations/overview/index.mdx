# Интеграции

Qwik поставляется со встроенными интеграциями, облегчающими подключение к вашим любимым сервисам, использование ваших любимых инструментов для создания стилей или тестирование и развёртывание вашего приложения в привычных сервисах, таких как Cloudflare, Netlify, Express server, и [Генератор статических сайтов (SSG)](https://qwik.builder.io/qwikcity/static-site-generation/static-site-config/).

Как только вы создадите приложение Qwik City, выполните следующую команду, чтобы добавить новые интеграции:

```shell
npm run qwik add
```

Вам будет предложено выбрать интеграцию, которую вы хотите добавить. Как только вы выберете интеграцию, она будет добавлена в ваше приложение, и вы сможете начать ее использовать.

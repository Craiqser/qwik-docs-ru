# `useLocation()`

Функция `useLocation()` возвращает объект `RouteLocation` для текущего маршрута.

```tsx
export interface RouteLocation {
  readonly params: RouteParams; // Record<string,string>
  readonly href: string;
  readonly pathname: string;
  readonly query: Record<string, string>;
}
```

Возвращаемое значение `useLocation()` аналогично `document.location`, но его безопасно использовать на сервере, где нет глобального объекта `location`.

## Параметры маршрута

`useLocation()` извлекает [параметры маршрута](../../../qwikcity/routing/route-parameters/index.mdx) в объект `params`.

Предположим, что у вас есть:

- Маршрут: `https://example.com/sku/[skuId]`
- Пользователь переходит к: `https://example.com/sku/1234`
- Затем `skuId` можно получить через `useLocation().params.skuId`

```typescript
export default component$(() => {
  const { pathname, params } = useLocation();

  return (
    <>
      <h1>Артикул</h1>
      <p>Путь: {pathname}</p>
      <p>Код: {params.skuId}</p>
    </>
  );
});
```

Приведенный выше код будет генерировать:

```html
<h1>Артикул</h1>
<p>Путь: /sku/1234</p>
<p>Код: 1234</p>
```

# Интеграции

Qwik поставляется со встроенными интеграциями, облегчающими подключение к вашим любимым инструментам и сервисам.

Как только вы создадите приложение Qwik City, выполните следующую команду, чтобы добавить новые интеграции:

```shell
npm run qwik add
```

Вам будет предложено выбрать интеграцию, которую вы хотите добавить. Как только вы выберете интеграцию, она будет добавлена в ваше приложение, и вы сможете начать ее использовать.

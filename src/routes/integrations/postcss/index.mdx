# PostCSS

PostCSS позволяет разработчикам писать современный CSS, сохраняя при этом обратную совместимость и повышая производительность. Он анализирует CSS-код и пропускает его через серию плагинов, которые могут изменять, оптимизировать и расширять CSS, прежде чем вывести конечный результат. [Сайт PostCSS](https://postcss.org/).

## Использование

Вы можете добавить PostCSS в свой проект с помощью следующего стартового шаблона Qwik:

```shell
npm run qwik add postcss
```

Предыдущая команда обновляет ваше приложение и устанавливает корректную конфигурацию в файле `vite.config.ts`.

Также она добавляет новый файл `postcss.config.js` в папку `src`.

# LeafletJS Map

Leaflet — это ведущая JavaScript-библиотека с открытым исходным кодом для создания интерактивных карт, удобных для мобильных устройств. Имея размер около 42 КБ JS, она обладает всеми функциями карт, которые когда-либо нужны большинству разработчиков.

Библиотека Leaflet разработана с учётом простоты, производительности и удобства использования. Она эффективно работает на всех основных настольных и мобильных платформах, может быть расширена с помощью множества плагинов, имеет красивый, простой в использовании и хорошо документированный API, а также простой, читаемый исходный код, вносить свой вклад в который приятно. [Веб-сайт LeafletJS Map](https://leafletjs.com/)

## Использование

Вы можете легко добавить LeafletJS Map, используя следующий стартовый шаблон Qwik:

```shell
npm run qwik add leaflet-map
```

Предыдущая команда добавляет в ваше приложение необходимые зависимости.

- `leaflet@1.9.4`
- `@types/leaflet@1.9.4`

Она также добавляет новые файлы в папку вашего проекта:

- `src/helpers/boundary-box.tsx`: Функция проверки границ области карты.
- `src/models/location.ts`: Модель для определения элементов информации о местоположениях, которые будут использоваться в параметрах.
- `src/models/map.ts`: Модель для определения информации о карте для использования в параметрах.
- `src/components/leaflet-map/index.tsx`: Простой компонент карты.
- `src/routes/basic-map/index.tsx`: Пример использования компонента Leaflet Map с демонстрационными данными.

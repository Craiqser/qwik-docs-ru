# Интеграции

Qwik поставляется со встроенными интеграциями, облегчающими подключение к вашим любимым инструментам и сервисам.

Как только вы создадите приложение Qwik City, выполните следующую команду, чтобы добавить новые интеграции:

```shell
npm run qwik add
```

Вам будет предложено выбрать интеграцию, которую вы хотите добавить. Как только вы выберете интеграцию, она будет добавлена в ваше приложение, и вы сможете начать ее использовать.

## Добавление новой интеграции

Спасибо за интерес к добавлению интеграции в Qwik! Мы будем рады помочь вам начать работу. Во-первых, если вы хотите добавить развёртывание сервера, ознакомьтесь с документацией [Server Deployments](/docs/deployments/index.mdx#добавить-новое-развёртывание). Если вы хотите добавить библиотеку Qwik, обратитесь к документации [Library](/docs/(qwik)/advanced/library/index.mdx). Если вы хотите добавить новую интеграцию инструментов или услуг, читайте дальше.

# Orama

[Orama](https://oramasearch.com/) - это быстрая полнотекстовая поисковая система, полностью написанная на TypeScript и не имеющая зависимостей.

Orama может быть использована в Qwik как на стороне клиента, так и на стороне сервера с помощью функций `routeLoader$`, `routeAction$` и `server$`.

Самый простой способ добавить Orama в Qwik - использовать команду `pnpm qwik add orama`. В результате будут установлены необходимые зависимости и создан новый публичный маршрут `/src/routes/orama` для демонстрации интеграции с Orama.

```bash
npm run qwik add orama
```

> Orama была разработана для работы с любой средой исполнения и не имеет зависимостей. Единственным требованием является наличие среды выполнения JavaScript.

Более подробную информацию можно найти в [документации Orama](https://docs.oramasearch.com/).

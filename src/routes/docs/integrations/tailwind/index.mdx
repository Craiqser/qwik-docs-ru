# Tailwind

Tailwind — это CSS-фреймворк, предоставляющий нам служебные классы узкого назначения, которые помогают нам стилизовать наши веб-страницы прямо внутри нашего файла разметки или файлах с расширениями `.js/.jsx/.ts/.tsx/.mdx` ([сайт Tailwindcss](https://tailwindcss.com/)).

## Использование

Вы можете легко добавить Tailwind с помощью следующего стартового шаблона Qwik:

```shell
npm run qwik add tailwind
```

Предыдущая команда обновляет ваше приложение и необходимые зависимости.

Также она добавляет новые файлы в директорию проекта:

- `postcss.config.js`
- `tailwind.config.js`
- `.vscode/settings.json`

и изменяет ваш файл `src/global.css`, чтобы включить в него

```css title="src/global.css"
# global.css file

@tailwind base;
@tailwind components;
@tailwind utilities;

...stuff...

```

# Устаревшие методы Qwik

Эти методы были объявлены устаревшими при подготовке стабильной версии Qwik.

Если вы случайно столкнулись с одним из них и не знаете, чем его заменить, мы упомянем их в этой таблице со ссылкой на их новую документацию ✨.

## Устаревшие методы и варианты их замены

---

| Устаревший метод | Замена |
| :-- | :-- |
| `createContext` | [createContextId](../components/context/#createcontextid) |
| `useBrowserVisibleTask` - `useBrowserVisibleTaskQrl` | [useVisibleTask$](../../(qwikcity)/route-loader/#routeloader) |
| `useClientEffect` - `useClientEffectQrl` | [useVisibleTask$](../../(qwikcity)/route-loader/#routeloader) |
| `useClientMount` | [useTask$](../components/tasks/#usetask) + [isBrowser](../../(qwikcity)/guides/qwik-nutshell/#isbrowser-conditional)
| `useEnvData` | [useServerData](../../../api/qwik/#useserverdata) |
| `useMount$` | [useTask$](../components/tasks/#usetask) |
| `useRef` | [useSignal](../components/state/#usesignal) |
| `useServerMount` | [useTask$](../components/tasks/#usetask) + [isServer](../../(qwikcity)/guides/qwik-nutshell/#isserver-conditional)
| `useWatch$` | [useTask$](../components/tasks/#usetask) |

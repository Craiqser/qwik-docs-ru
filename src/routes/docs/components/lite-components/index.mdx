# Легковесные компоненты

В дополнение к стандартным Qwik-компонентам, обладающим всеми свойствами отложенной загрузки, Qwik также поддерживает легковесные (`lite`) компоненты, которые больше похожи на традиционные фреймворки.

```tsx
// Легковесный компонент: объявлен с использованием стандартной функции.
export const MyButton = (props: { text: string }) => {
  return <button>{props.text}</button>;
};

// Компонент: объявлен с использованием функции `component$()`.
export const MyApp = component$(() => {
  return (
    <div>
      Немного текста:
      <MyButton text="Нажми меня" />
    </div>
  );
});
```

В приведённом выше примере `MyButton` является легковесным компонентом. Ленивая загрузка легковесных компонентов объединяется с обычным компонентом, частью которого они являются. И в таком случае:

- `MyButton` будет связан с функцией рендера `MyApp`.
- Всякий раз, когда выполняется функция рендера `MyApp`, она также вызывает выполнение рендера `MyButton`.
- `MyButton` не имеет владельца.

Вы можете представлять легковесные компоненты как блоки, встроенные в компонент, в котором они создаются.

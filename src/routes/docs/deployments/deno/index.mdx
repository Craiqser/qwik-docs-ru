# Промежуточное ПО для Deno

Промежуточное ПО Qwik City Deno позволяет вам подключить Qwik City к серверу Deno, который использует стандартные запросы/ответы, хорошо поддерживаемые Deno. Некоторые реализации сервера Deno включают:

- [Встроенный HTTP-сервер Deno](https://deno.com/manual/examples/http_server)
- [Oak](https://oakserver.github.io/oak/)

## Установка

Чтобы интегрировать адаптер `deno`, используйте команду `add`:

- Для [встроенного HTTP-сервера](https://deno.com/manual/examples/http_server):

```shell
npm run qwik add deno
```

## Сборка для производственной среды

Чтобы собрать приложение для производственной среды, используйте команду `build`. Она автоматически запустит `npm run build.server` и `npm run build.client`:

```shell
npm run build
```

## Запуск

Чтобы запустить сервер Deno после сборки, используйте команду `serve`:

```shell
npm run serve
```

## Развёртывание в производственной среде

Так как вы выбрали Deno, вы находитесь в собственной среде после запуска `npm run build`:

- Будет создана папка `dist`, включающая все статические файлы.
- Будет создана папка `server`, включающая все файлы сервера.

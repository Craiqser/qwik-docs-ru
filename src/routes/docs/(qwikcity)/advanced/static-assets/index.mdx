# Статические ресурсы

Используйте каталог `/public` для хранения всех статических ресурсов, таких как изображения, шрифты и т.д.

Путь к статическим ресурсам вычисляется относительно корневой директории вашего сервера (`/` в `https://example.com/`) и накладывается на ваши маршруты (маршруты имеют приоритет).

```bash
public/
├── favicon.ico           # http://example.com/favicon.ico
└── images/
    └── logo.png          # http://example.com/images/logo.png
```
